Spider Paint 3D experimental scene builder
3D framework: three.js
USE:: click+ALT Cylinder to surface
click+CTRL (2 times on object): draw line
speed paint: key C ( draw cylinder - need to curson on other surface ), delete: key X
A - Rotate, S - Zoom, D - Pan +(mouse)
E - draw plain with brush texture ( now around origo )
V - selected object start anim
R - animated objects reset to origo
T - tick animation
